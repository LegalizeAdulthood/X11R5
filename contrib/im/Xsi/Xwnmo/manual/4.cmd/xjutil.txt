


XJUTIL(4)           UNIX Programmer's Manual            XJUTIL(4)



名称
     xjutil - 辞書ユーティリティ・マネージャ

形式
     //uussrr//bbiinn//xxjjuuttiill ディスプレイ名 アトム名 カウンタ

解説
     _X_j_u_t_i_l は、 _W_n_n の辞書を操作するための辞書ユーティリティ・マ
     ネージャです。これは、 _x_w_n_m_o より自動的に起動されます。

操作
     _X_j_u_t_i_l は、 _W_n_n の辞書に対して、以下の操作を行ないます。

     辞書追加 ::
             現在の環境に新しい辞書を追加します。

     辞書一覧 ::
             現在使用している辞書の一覧を表示します。

     登録 ::  個人辞書に対して登録を行ないます。

     検索 ::  読みによる検索を行ないます。また、表示されたエントリ
             でボタンを押すことにより、そのエントリに対して、操作
             が行なえます。 ( 削除 / 使用不可 / 使用可 / コメント
             )

     パラメータ変更 ::
             現在の変換環境のパラメータを変更します。

     辞書情報 ::
             現在使用している辞書の詳しい情報を表示します。また、
             表示されたエントリでボタンを押すことにより、そのエン
             トリに対して、操作が行なえます。 ( 削除 / 使用不可 /
             使用可 / コメント )

     附属語変更 ::
             附属語ファイルを変更します。

関連項目
