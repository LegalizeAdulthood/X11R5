かな漢字変換の終了処理はKC\_FINALIZEを行います。
先のサンプルプログラムでは31行目で呼び出されています。

終了処理では以下のこ
とが行われます。

\begin{enumerate}
  \item 学習情報の辞書への書き込み
  \item 辞書のアンマウント
  \item かな漢字変換サーバとの接続の切断
  \item ローマ字かな変換テーブルの解放
  \item 各種カスタマイズ設定のリセット
  \item かな漢字変換に使われていたメモリの解放
\end{enumerate}

KC\_FINALIZE の時の第３引数の取り扱いは KC\_INITIALIZE のときと同じにな
ります。

終了処理はプログラムが終了する時はもちろん、かな漢字変換の使用を終了し
た時は必ず呼び出して下さい。
